# Materiale privato locale

Questa cartella può contenere note, configurazioni e documenti di supporto non destinati al versionamento.

Tutto il contenuto, eccetto questo file, è escluso da Git. Non usare questa cartella per esempi pubblici o file necessari all'avvio dell'applicazione.

Per un database caricato automaticamente via HTTP usare `data/user/`; per l'uso diretto `file://` conservare backup JSON in una posizione scelta dall'utente. I flussi sono descritti in [`docs/GESTIONE-DATABASE.md`](../../docs/GESTIONE-DATABASE.md).
